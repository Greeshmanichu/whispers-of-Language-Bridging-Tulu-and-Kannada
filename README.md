Project Description
Whispers of Language Bridging: Tulu and Kannada
Developed an intelligent language-bridging system for recognizing handwritten Tulu characters and translating them into Kannada text. The project utilizes a custom-built dataset containing Tulu vowels, consonants, and conjunct characters, along with deep learning-based character recognition models. A real-time application was developed to identify handwritten Tulu input and accurately convert it into the corresponding Kannada script using a predefined character mapping mechanism.
Key Features
Built a comprehensive handwritten Tulu character dataset containing vowels, consonants, and conjunct characters.
Trained deep learning models for accurate handwritten Tulu character recognition.
Implemented a real-time translation pipeline to convert recognized Tulu characters into Kannada text.
Used character mapping (tulu_to_kannada_mapping.json) for seamless script conversion.
Developed an interactive application (tulu_app.py) for user-friendly handwritten input recognition and translation.
