# 🌉 Whispers of Language: Bridging Tulu and Kannada

A deep learning project for recognizing handwritten **Tulu characters** and translating them into their corresponding **Kannada characters**. This project aims to support the preservation of the Tulu language by leveraging modern AI and computer vision techniques.

## 🚀 Features

- Handwritten Tulu character recognition
- Tulu to Kannada character mapping
- Image preprocessing and augmentation
- Multiple deep learning models (CNN, DenseNet, ViT, GRU, LSTM, Seq2Seq)
- Model performance evaluation using accuracy, precision, recall, and F1-score

## 🛠️ Tech Stack

- Python
- PyTorch
- OpenCV
- NumPy
- Pandas
- Matplotlib
- Scikit-learn

## 📂 Dataset

The dataset contains **more than 1 lakh handwritten Tulu character images**, organized into training and testing folders. Each character is mapped to its corresponding Kannada character using a JSON mapping file.
